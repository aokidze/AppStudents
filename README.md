# Student Database Management System

Проект представляет собой удобное приложение для работы с базой данных студентов и преподавателей. Это решение было создано с целью предоставления быстрого и простого способа для добавления, просмотра и удаления данных о студентах и преподавателях с использованием красивого и интуитивно понятного интерфейса.

## Функциональные возможности

1. **Добавление студентов и преподавателей**  
   Простой и быстрый способ добавить информацию о студентах и преподавателях в базу данных. Интерфейс позволяет вводить все необходимые данные через удобные формы.

2. **Просмотр данных**  
   Легкий доступ к данным студентов и преподавателей. Вы можете просматривать всю информацию в таблице, отфильтровав и отсортировав её по различным критериям.

3. **Удаление данных**  
   Удаление записей о студентах и преподавателях из базы данных с одним кликом.
